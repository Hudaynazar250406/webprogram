РЕАЛИЗОВАННЫЕ ВОЗМОЖНОСТИ PHP
================================================================================

1. Инициализация переменных:
   - $start_value - начальное значение аргумента (-10)
   - $encounting - количество вычисляемых значений (30)
   - $step - шаг изменения аргумента (1)
   - $min_value - минимальное значение для остановки (-50)
   - $max_value - максимальное значение для остановки (100)
   - $type - тип верстки ('A', 'B', 'C', 'D', 'E')

2. Вычисление функции (Вариант 16):
   При x <= 0:       f(x) = x^2 / 10
   При 0 < x < 15:   f(x) = (x + 5) / (x - 10)
   При x >= 15:      f(x) = sqrt(x - 14) + 2

3. Обработка ошибок:
   - Деление на ноль при x = 10 (вторая ветка)
   - Округление до 3 знаков после запятой (функция round())
   - Вывод "error" при невозможности вычисления

4. Типы верстки (переменная $type):
   'A' - Простая текстовая верстка с <br>
   'B' - Маркированный список <ul>
   'C' - Нумерованный список <ol>
   'D' - Табличная верстка <table>
   'E' - Блочная верстка <div> с красной рамкой

5. Статистические вычисления:
   - Максимальное значение функции
   - Минимальное значение функции
   - Среднее арифметическое
   - Сумма всех значений

6. Три типа циклов (в отдельных файлах):
   - index.php: цикл FOR с оператором break
   - index_while.php: цикл WHILE с предусловием
   - index_dowhile.php: цикл DO-WHILE с постусловием

================================================================================
ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ
================================================================================

1. Для проверки HTML-шаблона:
   - Откройте index.html в браузере
   - Проверьте на validator.w3.org

2. Для запуска PHP-скрипта:
   - Разместите файлы на веб-сервере с поддержкой PHP
   - Откройте index.php в браузере
   - Для смены типа верстки измените значение переменной $type

3. Для тестирования разных циклов:
   - index.php - цикл FOR (рекомендуемый)
   - index_while.php - цикл WHILE
   - index_dowhile.php - цикл DO-WHILE

================================================================================
ПРИМЕЧАНИЯ
================================================================================

- Функция для варианта 16 выбрана с учетом различных математических операций
- При x = 10 происходит деление на ноль, выводится "error"
- Ограничения min/max прекращают вычисления при выходе за пределы
- Цикл FOR наиболее оптимален для данной задачи (известно количество итераций)
- Цикл WHILE удобен при неизвестном заранее количестве итераций
- Цикл DO-WHILE гарантирует выполнение хотя бы одной итерации
