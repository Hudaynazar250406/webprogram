================================================================================
                         РЕАЛИЗОВАННЫЕ ТРЕБОВАНИЯ
================================================================================

1. СТАТИЧЕСКИЙ САЙТ:
   ✓ 3 страницы с одинаковым макетом
   ✓ Меню в виде ссылок в шапке страницы
   ✓ Ссылка на текущую страницу выделена стилем CSS
   ✓ Заголовок страницы (title): ФИО, группа, номер и название лабораторной
   ✓ Шапка (header): фиксированная, темно-зеленый фон, белые буквы
   ✓ Подвал (footer): фиксированный, темно-серый фон, светло-серые буквы
   ✓ Один H1 и не менее двух H2 на каждой странице
   ✓ Текст не менее 1Кб на каждую страницу
   ✓ Таблица с 2 строками и 3 колонками на каждой странице
   ✓ Не менее 2 фотографий на каждой странице

2. ДИНАМИЧЕСКИЙ КОНТЕНТ (PHP):
   
   a) Название страницы (TITLE):
      - Сохраняется в переменной $page_title
      - Выводится с помощью echo $page_title
   
   b) Подвал:
      - Выводит актуальную дату и время
      - Формат: "Сформировано DD.MM.YYYY в HH-MM-SS"
      - Используется функция date()
   
   c) Таблицы:
      - Первая строка (заголовки) полностью выводится PHP
      - Вторая строка: только содержимое ячеек через PHP
      - Используется echo для формирования HTML-кода
   
   d) Меню:
      - ДВА включения PHP-кода для каждого пункта меню
      - Первый фрагмент: выводит href и начало тега
      - Второй фрагмент: выводит класс (если активная) и текст ссылки
      - Одинаковая структура для всех пунктов, различаются только значения переменных
   
   e) Фотографии:
      - В зависимости от секунды (четная/нечетная) загружаются разные фото
      - Используется date('s') % 2 для определения четности
      - Две пары фотографий: foto1.jpg и foto2.jpg

================================================================================
                              ЗАПУСК ПРОЕКТА
================================================================================

1. Установите локальный сервер (XAMPP, WAMP, OpenServer или аналогичный)

2. Поместите папку crypto-counter в директорию веб-сервера:
   - Для XAMPP: htdocs/crypto-counter/
   - Для WAMP: www/crypto-counter/
   - Для OpenServer: domains/crypto-counter/

3. Запустите веб-сервер (Apache с поддержкой PHP)

4. Откройте браузер и перейдите по адресу:
   http://localhost/crypto-counter/

================================================================================
                            ТЕХНИЧЕСКИЕ ДЕТАЛИ
================================================================================

Использованные PHP-функции:
- date() - для получения текущей даты и времени
- echo - для вывода динамического контента
- Условный оператор if-else - для определения активного пункта меню и выбора фото
- Оператор % (остаток от деления) - для определения четности секунды

Переменные:
- $page_title - название страницы
- $menu_name - текст пункта меню
- $menu_link - ссылка пункта меню
- $current_page - флаг активности пункта меню
- $photo_name - имя файла фотографии
